# practis - Automation Assignment

### Requirements:
- Maven 3.8.6
- Java JDK 18.0.1.1
- Chrome browser

### Setup:
- git clone https://github.com/automatedguy/practis.git
- mvn clean test

### Dependencies:
- Selenium: Web driver libraries
- TestNG: annotations and asserts
- Webdrivermanager: auto setup for chrome driver
